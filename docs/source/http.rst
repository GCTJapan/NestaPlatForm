.. _http:


====================
HTTP Access
====================

NESTA Platform APIのアクセス方法を示します。

------------------------------------------------------------------------------

APIセットアップ
====================

NESTA Platform運営チームから発行された認証トークン(Bearer Token)を用意します。

.. code-block::

    DazDL7Ey/bRH8dVBHSi03g==  (認証トークン例)

------------------------------------------------------------------------------

認証方法
===================

headerに認証トークンをセットします。

.. code-block::

    "Authorization: Bearer DazDL7Ey/bRH8dVBHSi03g==" 

------------------------------------------------------------------------------

エンドポイント
===================

======================== ===============================
環境                     Endpoint                      
======================== ===============================
本番環境エンドポイント   |url_prod|
テスト環境エンドポイント |url_stg|
======================== ===============================

------------------------------------------------------------------------------

Content-Type
===================

.. code-block::

    "Content-Type: application/json"

------------------------------------------------------------------------------

実行例
===================

.. code-block:: 

    curl -X POST "http://pf-stg.nesta.world/api/user" -H  "Authorization: Bearer DazDL7Ey/bRH8dVBHSi03g==" -H  "Content-Type: application/json" -d "{\"customer_id\":\"customer_123\"}"
